# sicss2026
This code forms the basis for a research paper presented at the 2026 Summer Institute in Computational Social Science at the University of Rochester. Paper authored by Shea Thorpe, Nathan Carpenter-Holmes, and Mateo Santalucia.

We aim to quantify market response to major events occurring during the development of the Stargate data center in Abilene, Texas. We target three sectors: Electrical equipment manufacturers, power utility suppliers, and small modular reactor providers. 

We gather ticker price data dating back to 2023, and run OLS regressions over a predefined estimation window to determine Fama-French three- and five-factor betas. We define event windows and compute expected return during event windows. 

We compare expected return to actual return, in search of abnormal return. Our research paper contains our findings. Or you can run the code for yourself.
